# Customize the drumkit


## create a customize panel: 
    It should have customize toggle button of on and off
    - When customize toggle button is on
        Edit keys button should be enabled
        Delete Keys button should be enabled
        Default will also have a toggle button which will be off

    - When customize toggle button is off
        Edit keys and delete keys button should be disabled
        Default button will be enabled...indicating that the drumkit is  in default settings.
